Overview

ReadScore is a Chrome extension that analyzes web page readability, reading level, and text complexity using transparent, rule-based readability formulas. It provides instant readability scores, highlights hard-to-read sentences and dense paragraphs, and helps improve content clarity and writing quality. All analysis runs locally in your browser.
